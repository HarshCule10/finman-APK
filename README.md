# finman-APK
Hey, This is my submission for the assignment of SWE(IOS/ANDROID) role.
Setup->
Download and install the apk.
Enter basic details and you are good to go. I would recommend setting up a pin.

Requirements -> All the Core requirements mentioned in the document have been covered (Expenses,Income,Categories,Monthly summary,Dark/Light mode).
Added Features -> I worked on adding the following features
1. PIN based authentication for the app (as its a offline app, having email pw fields didnt make sense).
2. Built Recurring Income/Expenses, Editable from the profile.
3. Hidden Transactions. You can hide a transaction and store it in a vault only accessible by a pin (a pin should be setup for this to work).
4. Built a search tab, where extensive filtering options are available.
